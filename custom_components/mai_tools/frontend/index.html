✏️ Nhập Long-Lived Access Token
Long-Lived Access Token
Tạo token tại: Hồ sơ người dùng HA → Long-Lived Access Tokens
Token được lưu trong localStorage của trình duyệt, không gửi ra ngoài.
📖 Hướng dẫn tạo token
  1. Đăng nhập vào Home Assistant
  2. Nhấn vào avatar / tên tài khoản ở góc dưới bên trái sidebar
  3. Cuộn xuống cuối trang → mục Long-Lived Access Tokens
  4. Nhấn Create Token, đặt tên (vd: M.A.I Tools)
  5. Sao chép token và dán vào ô nhập ở trên
⚠️
Token này có quyền truy cập đầy đủ vào Home Assistant. Không chia sẻ token cho người khác. Nếu lộ token, hãy xóa ngay tại trang Hồ sơ HA.
